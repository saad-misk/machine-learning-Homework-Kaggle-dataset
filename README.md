# Student Academic Performance Prediction

## Project Overview
This project compares some machine learning algorithms (Logistic Regression, Decision Tree and AdaBoost) for predicting whether a website is benign or malicious.

## Dataset
- **Source:** Kaggle - "dataset-of-malicious-and-benign-webpages"
- **Target Variable:** `label` (discrete)
- **Features:** 12 features including url, url_length, ip, js_length, etc.

## Algorithms Compared
1. **Logistic Regression**: A simple, interpretable linear model used for binary classification that estimates probabilities using a logistic function.
2. **Decision Tree**: A tree-based model that splits data into branches based on feature values to make clear, rule-based predictions.
3. **AdaBoost**: An ensemble boosting method that combines multiple weak learners sequentially, focusing on correcting previous errors to improve overall accuracy.
