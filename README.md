# Student Academic Performance Prediction

## Project Overview
This project compares two machine learning algorithms (Linear Regression and Random Forest) for predicting student academic performance based on their habits and demographics.

## Dataset
- **Source:** Kaggle - "Students Habits VS Academic Performance"
- **Target Variable:** `exam_score` (continuous)
- **Features:** 15 features including study hours, social media usage, sleep patterns, attendance, etc.

## Algorithms Compared
1. **Linear Regression** - Simple, interpretable linear model
2. **Random Forest Regressor** - Ensemble method with multiple decision trees
